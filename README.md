# ☕ Coffee Corner – Café Website

## Overview

Coffee Corner is a modern and responsive café website designed to provide users with a smooth online café experience. Users can explore the menu, browse products, manage their cart, and access account features through a clean and user-friendly interface.

---

## Features

* Responsive modern UI/UX
* Homepage with attractive hero section
* Dynamic Menu Page
* Product filtering by category
* Add to Cart functionality
* Cart badge updates
* User Registration & Login
* Local Storage integration
* Mobile responsive navigation
* Footer with contact details

---

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Local Storage
